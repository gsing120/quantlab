# Open Questions

Questions that need human decision but aren't blocking current work. Claude Code adds to this file when it encounters ambiguity and proceeds with a conservative default. Review weekly.

Format: each entry is a question with context, the default being used, and who/when resolved.

## Template

```markdown
### Q-NNN — Short question

**Raised:** YYYY-MM-DD by [Claude Code / human]
**Context:** Why this came up.
**Default chosen:** What we're doing until resolved.
**Resolution:** (fill in when answered) Decision + date + link to ADR if architectural.
```
